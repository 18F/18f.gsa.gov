Developer Resources

We hope that developers everywhere make use of the APIs we build in our projects. Over time, you will find a growing number of APIs added to our developer, along with further resources to make their use easier.

FBOpen

The FBOpen API provides an interface to allow easier access to a range of business opportunities with the federal government. Learn more from the launch annoucnement and the GitHub repository.

API Standards

The 18F API Standards is a living document that serves as internal guidance for our developers as they build web services. We also want our partners to better anticipate how 18F projects operate. Please suggest improvements as we are always looking to improve.

Open Source Policy

The 18F Open Source Policy dictates our procedures when developing projects - both in how our team operates and how it interacts with the wider community. We believe that robust and open code management makes for better products. This policy is a living document, so please suggest any improvements.

/Developer Program

The /Developer Program is a suite of tools and resources to assist in the production and management of government APIs. The information is available to all and can be readily repurposed to the needs of individual agencies. Furthermore, the program includes various options for direct support which are availble to any federal agency.
