Developer Resources

We hope that developers everywhere make use of the APIs we build for our projects. Over time, you'll find a growing number of APIs listed here, along with resources and documentation to assist with their use.

18F's APIs

FBOpen

The FBOpen API provides an interface to allow easier access to a range of business opportunities with the federal government. Learn more from the launch announcement and the GitHub repository.

18F Initiatives

/Developer Program

The /Developer Program is a suite of tools and resources to assist in the production and management of government APIs. The information is available to all and can be readily repurposed to the needs of individual agencies. Furthermore, the program includes various options for direct support which are available to any federal agency. Learn more from the blog post announcing the program.

API.Data.gov

API.Data.gov is a free API management service for federal agencies. It provides an API with analytics, a proxy layer, and an API key solution.

API Standards

The 18F API Standards is a living document that serves as internal guidance for our developers as they build web services. We also want our partners to better anticipate how 18F projects operate. Please suggest improvements as we are always looking to improve our practices.
